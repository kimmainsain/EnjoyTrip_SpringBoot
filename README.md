| :----: | -------------------------------- | -------------------------------------------------------------------------------------------------------------------------- | :-----------: |
|  기본  | 메인 페이지 및 메뉴 구성         |                                                                                                                            |       O       |

|  기본  | 로그인/로그아웃 페이지           |                                                                                                                            |       O       |
|  추가  | 사이트맵/전체적인 메뉴 구성 화면 |                                                                                                                            |       O       |
|  심화  | 게시판 페이지                    | 글등록, 목록, 조회, 수정 화면                                                                                              |       O       |
